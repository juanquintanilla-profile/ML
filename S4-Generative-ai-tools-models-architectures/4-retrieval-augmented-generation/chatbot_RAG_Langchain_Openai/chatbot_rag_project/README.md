# Chatbot RAG con LangChain y OpenAI

Proyecto ejemplo: Chatbot RAG (Retrieval-Augmented Generation) con CLI.

Contenido:
- main.py: CLI entrypoint.
- core/rag_system.py: indexación y recuperación (InMemoryVectorStore simple).
- core/chatbot.py: lógica conversacional con mantenimiento de contexto.
- documents/documento1.md, documento2.md: contenidos de ejemplo (>=500 palabras cada uno).
- requirements.txt: dependencias.
- .env.example: ejemplo de variables de entorno.

Sigue las instrucciones en .env.example antes de ejecutar el proyecto.
